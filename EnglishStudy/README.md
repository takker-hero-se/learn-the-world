# 英語ワードリスト（NGSL関連）

New General Service List Project（https://www.newgeneralservicelist.com）から取得した、英語学習者向けの頻出語彙リスト集です。すべてバージョン1.2（最新版）です。

## フォルダ構成

### NGSL/（New General Service List — 一般英語 約2,800語）
日常会話や一般的な英文の約92%をカバーする基本語彙リストです。

| ファイル名 | 形式 | 説明 |
|---|---|---|
| NGSL_12_stats.csv | CSV | 頻度統計付きリスト |
| NGSL_12_alphabetized_description.txt | TXT | アルファベット順・説明付き |
| NGSL_12_lemmatized_for_teaching.csv | CSV | 教育用レンマ化リスト |
| NGSL_12_lemmatized_for_research.csv | CSV | 研究用レンマ化リスト |
| NGSL_12_with_English_definitions.xlsx | Excel | 英語定義付きリスト |
| NGSLwithSFI-31K.xlsx | Excel | 拡張頻度データ付きリスト |
| SUP_lemmatized.csv | CSV | 補助語リスト |

## ファイル形式の説明

- **stats**: 各単語の頻度順位や出現頻度などの統計データを含む
- **alphabetized_description**: アルファベット順に並べた単語と簡単な説明
- **lemmatized_for_teaching**: 教育目的で整理されたレンマ（見出し語）と活用形の対応表
- **lemmatized_for_research**: 研究目的でより詳細なレンマ化データを含む
- **definitions**: 各単語に平易な英語での定義を付与したリスト
