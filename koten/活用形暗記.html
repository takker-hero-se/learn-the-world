動詞の活用 9種類の覚え方

「よ　か　さ　な　ら　し　し　か　か」
四段・上一段・上二段・下一段・下二段・カ変・サ変・ナ変・ラ変
四(よ)つの段 → 上一(か・み)段 → 上二(か・み)段 → 下一(し・も)段 → 下二(し・も)段 → カ変 → サ変 → ナ変 → ラ変

四段活用の活用語尾

あ・い・う・う・え・え
未然「a」連用「i」終止「u」連体「u」已然「e」命令「e」
例: 書か・書き・書く・書く・書け・書け

上二段活用の活用語尾

い・い・う・うる・うれ・いよ
例: 起き・起き・起く・起くる・起くれ・起きよ

下二段活用の活用語尾

え・え・う・うる・うれ・えよ
例: 受け・受け・受く・受くる・受くれ・受けよ

カ行変格活用「来(く)」

こ・き・く・くる・くれ・こよ
「こない・きます・くる・くるとき・くれば・こい」と現代語で対応させて覚える

サ行変格活用「す」

せ・し・す・する・すれ・せよ
現代語「する」の活用とほぼ同じ！

ナ行変格活用「死ぬ・往(い)ぬ」

な・に・ぬ・ぬる・ぬれ・ね
「な行」を順番に: な・に・ぬ、ぬる・ぬれ・ね

ラ行変格活用「あり・をり・はべり・いまそかり」

ら・り・り・る・れ・れ
ラ変動詞は4語だけ!「あり・をり・はべり・いまそかり」
「あを(青)はべり いまそかり」で覚える

形容詞 ク活用の活用語尾

(く)から・く・し・き・けれ・○
カリ活用（未然・連用）: から・かり
例: 美しく(から)・美しく・美し・美しき・美しけれ

形容詞 シク活用の活用語尾

(しく)しから・しく・し・しき・しけれ・○
ク活用に「し」がつく！
例: 楽しく(しから)・楽しく・楽し・楽しき・楽しけれ

助動詞の接続 まとめ

「未然にムズムズ　さすらじまほし」
未然形接続: る・らる・す・さす・しむ・ず・じ・む・むず・まし・まほし

連用形接続: き・けり・つ・ぬ・たり(完了)・たし・けむ
→ 「き・けりつぬたり　たしけむ」

終止形接続: べし・まじ・らし・めり・なり(伝聞推定)
→ 「べし・まじらしめりなり」

連体形接続: なり(断定)・たり(断定)・ごとし
→ 断定の「なり・たり」とごとし

「る・らる」の4つの意味

受・尊・自・可
受身・尊敬・自発・可能
「受尊自可(じゅそんじか)」と音読みで覚える

「べし」の6つの意味

す・い・か・と・め・た
推量・意志・可能・当然・命令・適当
「すいかとめた」(スイカ止めた) で覚える！
